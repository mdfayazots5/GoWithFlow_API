## Module List

1. Project Documentation
   - Section: `Repository Structure`
   - Purpose: locate current repo areas and documentation ownership

2. Backend Authentication Foundation
   - Section: `Backend Authentication Foundation`
   - Purpose: .NET 8 API, SQL schema, stored procedures, JWT auth, OTP flow, repositories, controller endpoints

3. Backend User Module
   - Section: `Backend User Module`
   - Purpose: self-service profile, avatar upload, session detail, improvement analytics, streaks, and badges

4. Backend Admin Module
   - Section: `Backend Admin Module`
   - Purpose: admin dashboard, user management, admin notes, report summary, full report export, admin-only APIs

5. Backend Script Module
   - Section: `Backend Script Module`
   - Purpose: excel validation, script upload, script library retrieval, utterance storage, version history, sample template APIs

6. Backend Session Module
   - Section: `Backend Session Module`
   - Purpose: session creation, join code validation, lobby state, ready/start/end/leave flow, session history, SignalR lobby events

7. Backend Live Session Module
   - Section: `Backend Live Session Module`
   - Purpose: turn orchestration, re-read handling, voice analysis capture, listener feedback, session completion summary, SignalR live-session events

8. Backend Mistake Repractice Module
   - Section: `Backend Mistake Repractice Module`
   - Purpose: mistake persistence, mistake summary, grammar progress, repractice generation, repractice attempt tracking, and repractice completion APIs
